# ES6 Workflow

Use ES6, today.

### Install

The very first, project [6to5](https://6to5.org) is required.

```bash
[sudo] npm install --global 6to5
```

### Build

Build dist files from `./src` folder

```bash
npm run build
```

Build and concat to one minimal file.

```bash
npm run build-concat
```

### Watch and serve

```bash
npm run dev
```
